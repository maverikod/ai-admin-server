## 🎯 Обзор

Добавлены две новые команды для управления отдельными задачами в очереди: остановка (отмена) и удаление задач.

## 🆕 Новые команды

### 1. QueueCancelTaskCommand
- **Команда:** `queue_cancel_task`
- **Файл:** `ai_admin/commands/queue_cancel_task_command.py`
- **Функциональность:**
  - Отмена конкретной задачи по ID
  - Поддержка принудительной отмены (`force=True`)
  - Проверка статуса задачи перед отменой
  - Безопасная отмена с валидацией

### 2. QueueRemoveTaskCommand
- **Команда:** `queue_remove_task`
- **Файл:** `ai_admin/commands/queue_remove_task_command.py`
- **Функциональность:**
  - Полное удаление задачи из очереди
  - Поддержка принудительного удаления (`force=True`)
  - Удаление из всех структур данных очереди
  - Безопасное удаление с валидацией

## 🎯 Использование

### Примеры команд

```bash
# Отменить задачу (обычная отмена)
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_cancel_task", "task_id": "task-123"}'

# Принудительно отменить выполняющуюся задачу
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_cancel_task", "task_id": "task-456", "force": true}'

# Удалить задачу из очереди
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_remove_task", "task_id": "task-789"}'

# Принудительно удалить выполняющуюся задачу
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_remove_task", "task_id": "task-abc", "force": true}'
```
