## 🎯 Использование

### Примеры команд

```bash
# Получить статус очереди
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_status", "action": "status", "detailed": true}'

# Проверить здоровье очереди
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_health", "detailed": true}'

# Получить статистику за 24 часа
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_statistics", "time_range_hours": 24}'

# Фильтровать задачи по статусу
curl -X POST http://localhost:20001/cmd \
  -H "Content-Type: application/json" \
  -d '{"command": "queue_filter", "filter_status": "pending", "detailed": true}'
```
