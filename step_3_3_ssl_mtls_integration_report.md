## 🚀 Использование

Сервер теперь поддерживает следующие режимы запуска:

```bash
# HTTPS only режим
python -m ai_admin.server --ssl-mode https_only

# mTLS режим
python -m ai_admin.server --ssl-mode mtls

# Token authentication режим
python -m ai_admin.server --ssl-mode token

# Отключение SSL
python -m ai_admin.server --disable-ssl

# Отключение mTLS
python -m ai_admin.server --disable-mtls

# Отключение token аутентификации
python -m ai_admin.server --disable-token-auth
```
