- Обнаружены предупреждения о deprecated `datetime.utcnow()`
- Рекомендуется заменить на `datetime.now(datetime.UTC)`
- Не критично для функциональности, но требует обновления

### 2. Cryptography Warnings
- Предупреждения о deprecated свойствах cryptography
- Рекомендуется использовать `not_valid_after_utc` вместо `not_valid_after`
